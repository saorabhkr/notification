# Notification UI widget
A custom widget to enable notification.

# Calling notification 
  $.fn.notificationWrapper({
    notificationType: "info",
    template: $("#noti-p-wrap").html(),
    width: "200px",
    height: 90,
    notificationPosition: "centerBottom",
    whereToAppend: $("body"),
    autoHide : false
  });
